# wallet-teamcuya
Simple Wallet System

## Installation
```
git clone https://github.com/jccuya123/wallet-teamcuya.git
cd wallet-teamcuya
npm i
node index.js
```

## URLs
### 1.) Fund Transfer
```
http://localhost:3300/fund
```

### 2.) Check Balance
```
http://localhost:3300/check
```


### 3.) Deposit
```
http://localhost:3300/deposit
```


### 4.) Withdraw
```
http://localhost:3300/withdraw
```


### 5.) Bills Paymet
```
http://localhost:3300/fund
```
